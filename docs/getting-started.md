# Getting Started

## Prerequisites

- Python 3.8 or higher

## Installation

1. Clone this repository:
   ```bash
   git clone <repository-url>
   cd desktop-tutorial
   ```

2. Run the application:
   ```bash
   python src/main.py
   ```

## Running Tests

```bash
python tests/test_main.py
```
